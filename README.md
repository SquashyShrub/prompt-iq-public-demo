# PromptIQ

Current Version: v0.3.0-dev | Last Updated: 2026-05-22

PromptIQ is an AI-assisted prompt optimization platform that helps users transform weak prompts into structured, higher-performing prompts while teaching prompt engineering principles.

## Status

Phase 1: Complete  
Phase 2: Complete  
Phase 3: In Progress, mostly complete

## Current Capabilities

- Landing page UI foundation
- Weak prompt input workflow
- API route for prompt improvement
- Service-layer prompt optimization workflow
- Deterministic scoring system
- Before/After comparison UI
- Prompt improvement categories
- Techniques-used display
- Structured result format
- Frontend-to-API workflow wiring
- Placeholder optimization logic without live OpenAI calls

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Next.js API Routes
- OpenAI Responses API planned
- Vercel deployment planned

## Development Notes

PromptIQ currently uses deterministic placeholder optimization logic. Real OpenAI Responses API integration is planned after the frontend workflow, result display, and teaching feedback systems are stable.
